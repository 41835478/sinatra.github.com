Sinatra (file, base=Sinatra::Default)

Sugar to include a classic style app in a rackup.

This will eval the source into a Sinatra::Default class Example:

require ‘sinatra/base‘

map ’/foo’ do 

run Sinatra("foo.rb")

end

run Sinatra(“bar.rb“)
